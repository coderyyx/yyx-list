# yyx-list
基于ECMAscript 2015 的Array Api 封装的类似java的List集合
把平时业务中常用的方法封装到一起，主要目的简化业务代码，结构更清晰。
希望大家提出宝贵意见，使它更加完善。

let ins = new List(array);
